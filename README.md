# 99 имён Аллаха

Минималистичное Flask-приложение для изучения 99 имён Аллаха.

## Что есть внутри

- карточки с арабским именем, русским именем, переводом и объяснением;
- flip-анимация карточки;
- кнопка следующей карточки;
- случайная карточка;
- счётчик прогресса;
- режим викторины;
- адаптивный минималистичный интерфейс.

## Технологии

- Python
- Flask
- HTML
- CSS
- JavaScript

## Как запустить локально

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

После запуска открой:

```text
http://127.0.0.1:5000
```
